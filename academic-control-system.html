# Academic Control System

# Description

This is an academic management system project of a university. I was the technical leader and I worked on it for 3 years. Its objective is automate/modernize all the academic related processes of the university.

We started the project from scratch and since I was the technical leader, I was free to choose the technologies for this project so I decided to go for Laravel, Vue.js and MySQL since they were the technologies that I mastered, they were suitable for the project and my team had knowledge with PHP, Javascript and MySQL so I just needed to teach them about the frameworks. All the team was involved on design and planning decisions.

Since the university really needed the system as fast as possible, we developed it while being used and tested at the same time on real life scenarios by the university. I'm not going to lie, that was some really extreme programming but very worth it at the end and I'm grateful for all the things that I learned there.

The system isn't finished yet, it covers like 20% of all the planned things that the system will have but all the things listed here are used in production. All the information displayed is dummy data because, for obvious reasons, I'm not allowed to show the system with real data.

# Authentication/Authorization

For authentication it uses OAuth2 provided by Laravel Passport (opens new window) package, LDAP Active Directory (opens new window) and Office365 authentication API. For authorization it uses a mix of Laravel default authorization methods (policies, gates, etc.) and spatie/laravel-permission (opens new window) package for roles and permissions logic.

# School

This module manages all school related things like:

  • Students
  • Teachers
  • Groups
  • Grades
  • Student attendances
  • Reports
  • Timelines (academic years, exam/test periods, etc.)
  • Enrollments / Re-Enrollments / Applicants: these processes are done by the students themselves through a student portal developed by us that is also part of this system, unfortunately I don't have screenshots of it. Students can also check their grades and schedule there.

Week Planner and Attendance List tools are part of this module.
